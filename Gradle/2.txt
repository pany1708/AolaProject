5. gradle -q task:
  q: 代表quite模式, 它不会生成gradle的日志信息.

eg:

task hello {
    doLast {
        println 'Hello world!'
    }
}

// << 操作符是 doLast 的简单别称.
task hello << {
    println 'Hello world!'
}
